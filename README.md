# Battles of Laponia

Multiplayer game made with C/C++.

## Getting Started

Clone this repository:

```
https://github.com/kokubum/laponia.git
```

### Running

You can compile first:
```
g++ main.cpp -o random_name
```
Or just run the program:

```
./laponiaGame
```
